# P-club-induction-project-(Study Room slot booking UI)
This is a frontend-only web application that simulates a study room booking system.
Users can view rooms, check available time slots, book a slot, and cancel bookings.
The system prevents double booking and stores all bookings using browser localStorage.



->Features:

-Clean UI

-Display multiple study rooms with Location and Capacity.

-Show available time slots.

-Book a time slot.

-Prevent double booking.

-Persist bookings using localStorage.

-Booking Immediately Updates the UI state.

-An additional nav bar just for website looks .



->Technolgies Used:

-HTML

-CSS

-JAVA-SCRIPT

-Local Storage
